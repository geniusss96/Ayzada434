🔄 Визуализатор LRU Кэша

Реализация структуры фиксированного размера на базе Hash Map + Doubly Linked List

Как это работает?

  • Hash Map обеспечивает поиск узла по ключу за константное время O(1).
  • Двусвязный список упорядочивает элементы по новизне использования. Самый свежий элемент (MRU) всегда перемещается к HEAD, а самый старый (LRU) вытесняется у хвоста TAIL за O(1).
